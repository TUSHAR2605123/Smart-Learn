# Smart-Learn
Smart Learn is a web-based platform designed for students to learn, share knowledge, and collaborate in one place. It provides user-friendly features like authentication, profile management, student listings, blogging, and a student wall for sharing ideas—making learning more interactive and engaging.
<br/>
🚀 Features
<br/>
 User Authentication – Secure login and registration system.
 <br/>
 Profile Management – View and update student profiles.
 <br/>
 Student List – Access a list of all registered students.
 <br/>
 Blog Writing – Create, edit, and delete blogs to share knowledge.
 <br/>
 Student Wall – A space for students to share posts (like Instagram-style).
 <br/>
 Search Functionality – Search for blogs and students easily.
 <br/>
 My Modules – Access additional modules (admin restricted).
 <br/>
 Support – Contact support for assistance.
